# Scored checks and their daemons:
#   check_ssh.py      -> sshd         (port 22,  RSA key auth + command exec)
#   check_ftp.py      -> vsftpd       (port 21,  login / file read / write / hash)
#   check_dns.py      -> bind9/named  (port 53,  forward + reverse lookups)
#   check_smb.py      -> smbd / nmbd  (port 445, login / share read / write)
#   check_http.py     -> apache2/nginx (port 80,  connectivity + web content)
#   check_mysql.py    -> mysql        (port 3306, connect / read / write)
#   check_postgresql.py -> postgresql (port 5432, connect / read / write)
#   check_icmp.py     -> (no daemon)  ICMP ping -- stopping the host or blocking
#                                     ICMP fails this check; caught by shutdown/
#                                     reboot/iptables patterns in SCORING_PATTERNS
